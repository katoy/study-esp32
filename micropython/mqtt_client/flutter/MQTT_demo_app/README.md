## ビルドと実行方法

このアプリのビルド・実行手順は以下の通りです。

1. 依存パッケージのインストール
	```sh
	flutter pub get
	```

2. アプリの実行（エミュレータまたは実機）
	```sh
	flutter run
	```

3. ビルド（apk/ipa/webなど）
	- Android:  `flutter build apk`
	- iOS:      `flutter build ios`
	- Web:      `flutter build web`

詳細は [Flutter公式ドキュメント](https://docs.flutter.dev/) を参照してください。
